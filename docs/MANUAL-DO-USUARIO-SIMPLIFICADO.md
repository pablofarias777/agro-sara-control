# 3. Manual do Usuario Simplificado

Este manual foi feito para usuarios leigos, com instrucoes diretas para usar as principais funcoes do Agro Sara Control.

## 1. Para que serve o sistema

O Agro Sara Control ajuda a organizar a gestao da propriedade rural, permitindo:
- cadastrar culturas e safras;
- registrar despesas, vendas e insumos;
- definir metas;
- acompanhar resultados;
- exportar relatorios.

## 2. Como entrar no sistema

1. Abra o sistema no navegador.
2. Se ainda nao tiver conta, clique em **Cadastrar**.
3. Se ja tiver conta, informe e-mail e senha na tela **Login**.

## 3. Primeiro acesso (Setup)

No primeiro acesso, faca o cadastro inicial da propriedade:
1. Informe nome da propriedade.
2. Informe nome do produtor.
3. Salve para concluir o setup.

Sem esse passo, as demais telas ficam bloqueadas.

## 4. Como usar as principais funcionalidades

### 4.1 Culturas

Use para cadastrar o que e produzido (ex.: milho, feijao, mandioca).

Passos:
1. Acesse **Culturas**.
2. Clique em **Adicionar**.
3. Preencha nome e unidade.
4. Salve.

### 4.2 Safras

Use para controlar ciclos de producao por cultura.

Passos:
1. Acesse **Safras**.
2. Clique em **Adicionar**.
3. Selecione a cultura.
4. Informe data de inicio e fim (quando houver).
5. Salve.

### 4.3 Despesas

Use para registrar gastos da propriedade.

Passos:
1. Acesse **Despesas**.
2. Clique em **Adicionar**.
3. Informe data, categoria, descricao e valor.
4. Salve.

### 4.4 Vendas

Use para registrar faturamento.

Passos:
1. Acesse **Vendas**.
2. Clique em **Adicionar**.
3. Informe data, cultura, quantidade, valor e canal de venda.
4. Salve.

### 4.5 Insumos

Use para controlar estoque de itens da producao.

Passos:
1. Acesse **Insumos**.
2. Clique em **Adicionar**.
3. Informe nome, categoria, unidade e quantidade.
4. Salve.

### 4.6 Metas

Use para definir objetivos financeiros e produtivos.

Passos:
1. Acesse **Metas**.
2. Clique em **Adicionar**.
3. Escolha escopo (cultura ou safra).
4. Defina meta de faturamento, limite de gasto e/ou meta de quantidade vendida.
5. Salve.

### 4.7 Dashboard

Mostra o resumo geral da operacao:
- receitas;
- despesas;
- saldo;
- comparacoes por periodo;
- progresso das metas.

### 4.8 Exportar e relatorios

Use para gerar relatorios em arquivo.

Passos:
1. Acesse **Exportar e relatorios**.
2. Escolha periodo e filtros desejados.
3. Gere o arquivo `.xlsx`.

## 5. Edicao e exclusao de registros

- Para editar: abra o item e clique em **Editar**.
- Para excluir: use a opcao **Excluir** e confirme.

Atencao: exclusao pode afetar relatorios e historico.

## 6. Dicas para evitar erros

- Preencha datas e valores com atencao.
- Cadastre primeiro a cultura antes da safra.
- Revise dados antes de excluir registros.
- Mantenha login e senha em local seguro.

## 7. Problemas comuns

### Nao consigo entrar
- Verifique e-mail e senha.
- Tente redefinir ou recadastrar usuario (se permitido no ambiente).

### Sistema nao carrega dados
- Confira se backend e banco de dados estao ligados.
- Atualize a pagina.

### Erro ao exportar
- Verifique se existe dado no periodo selecionado.
- Tente reduzir filtros e gerar novamente.

## 8. Encerramento seguro

Ao terminar o uso:
1. Salve o que estiver pendente.
2. Clique em **Sair** para encerrar a sessao.
